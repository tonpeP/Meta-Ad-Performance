# Swggy-Data-Dashboard
Project Title :
Meta Ad Performance
Example :A Power BI dashboard providing a comprehensive visual analysis of Meta ad performance across Instagram and Facebook. It tracks how users interact with ads through clicks, shares, and comments  do by gender, age group, and city. The dashboard bridges engagement metrics with actual purchase behavior, converting raw interaction data into actionable conversion insights. 

Short Description :
This dashboard visualizes the full funnel of Meta ad performance  from initial impressions to final purchases. It answers how many users clicked on an ad, shared it, or commented on it (indicating product interest), and ultimately how many completed a purchase.
Key metrics include:
Purchase rate and engagement rate broken down by age group (15–55)
Gender based click, share, and comment analysis
City level performance to identify high converting regions
Ad format comparison across Video, Image, Post, and Stories ads

Tech Stack:
PowerBI Desktop - Main data visualization platform used for building the interactive report
power query - Data transformation and cleaning layer for reshaping and preparing raw data 
My SQL - Backend data source used to understand table relationships and query structured data
Excel  - Used for filtering, sorting, and importing CSV data as a pre-processing step
data modeling - Relationships established among tables like sales report, tier-wise performance, ratings, and top-selling products

data source :
kaggle
